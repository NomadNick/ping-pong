Ping-

Pong!

Enter a number up to 100, click the button, and for each number that is divisible just by three, the game will print 'Ping!'.
For each number that is divisible just by five, the game will print 'Pong!'.
For each number in the series that is divisible by both 3 and 5, the game will print 'Ping-Pong!'.
If none of those coditions exist, the game will simply print the number.
